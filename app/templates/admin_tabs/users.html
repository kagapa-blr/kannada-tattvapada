👤 User Management

Manage your registered users, edit details, and control admin privileges.

How to Use User Management
Managing Users:
  • View all registered users in the table below.
  • Edit a user by clicking the Edit button in the Actions column. Update username, email, phone, or admin privilege.
  • Delete a user by clicking the Delete button. Confirm deletion in the popup modal.
  • Ensure you have the necessary admin privileges to manage users.
Tips:
  • Use search or filters (if enabled) to quickly locate users.
  • Deleting a user is permanent. Double-check before confirming.
  • Hover over table rows for highlight effects, improving navigation.
