Delete Tatvapada

How to use this tab:
  • Select individual Tatvapada(s): Use the checkboxes under "Tatvapada Sankhyes" to pick specific entries.
  • Delete selected Tatvapada(s): Click the "Delete Selected" button after choosing Tatvapada(s).
  • Delete all Tatvapadas of an author: Use the "Delete All" button in the last column of each row.
  • Delete entire samputa: Enter the samputa number below and click "Delete Samputa".
  • Select All: Use the top-left "Select All" checkbox to quickly select all visible Tatvapadas.
  • All delete actions require confirmation via a modal to prevent accidental deletion.
  • Success and failure notifications will appear in pop-up modals after each operation.
